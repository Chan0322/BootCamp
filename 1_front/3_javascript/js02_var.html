변수 사용

변수 선언 규칙

  1. 대소문자 구분한다. aA aa 구분.
  2. 이름은 영문, $, _로 시작해야 한다. (숫자 등은 불가)
  3. 영문,$,_,숫자를 포함하여 생성 가능하다.
  4. 키워드나 예약어를 사용할 수 없다.

변수의 범위

  1. 전역 변수 : 다른 함수들에서 공용으로 사용할 수 있다.(값이 유지된다.)
  2. 지역 변수 : 함수나 객체 내부에 선언되고고 실행이 종료되면 사라진다.

변수 선언과 저장

  1. let, var, const 키워드를 사용하여 변수를 선언한다.
  2. 변수의 타입은 저장되는 데이터의 타입에 따라 결정된다.

Infinity, NaN

  1. Infinity:양의 무한대, 0으로 나누었을 때
  2. NaN(Not a Number) : 숫자가 아닌 값.
